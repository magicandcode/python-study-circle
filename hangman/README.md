# Hangman's Game

### Starter code for Python Study Circle meetup #1 April 15 2020

Set up a virtual environment to run the project code.

Requires Python 3.6 or later.

### Unix:
```
python3 -m venv venv
```
```
. venv/bin/activate
```

### Windows:
```
python -m venv venv
```
```
# cmd
venv\Scripts\activate.bat

# Powershell
venv\Scripts\activate.ps1
```
